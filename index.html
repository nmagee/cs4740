Course Description

Investigates the architectural foundations of the various cloud platforms, as well as examining both current cloud computing platforms and modern cloud research. Student assignments utilize the major cloud platforms. Prerequisite: CS 2150 with grade of C- or higher.

Cloud Computing has become an everyday term, but its meaning often remains unclear. For the purposes of this course, "cloud computing" makes up a specific way of creating, managing, and interacting with remote computing infrastructure and services.

We will discuss the technical, business, and practical reasons that cloud computing emerged, and the ways it has evolved in the last 15 years.

Tools for this course

Students in this course will be expected to use the following pieces of software on a weekly basis:

  • Twitch.tv - This will be our primary platform for real-time and recorded lectures. Create a Twitch.tv account and verify your email address in order to participate in chat.
  • Discord - Discord will serve as our hub for discussions, Q&A, and distributing extra material specific to topics or assignments. Create a Discord account and join the class server.
  • Git/GitHub - Students are expected to know how to version control their material using Git. Some programming assignments will require submission via commits pushed to GitHub. If you do not have a GitHub account, create one now.
  • AWS Console - All students will be given a pre-paid Amazon Web Services account for this course.
  • AWS CLI/SDKs - We will regularly interact with the AWS cloud programmatically. This will include the CLI interface and at least one SDK, preferably the boto3 library for Python.
