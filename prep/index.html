Preparing for the Fall

Learn the Linux shell and do not be afraid of it. We will be using the AWS command-line tools, will do some scripting in bash, and you will need to read and generate plenty of JSON and YAML. Bonus: Learn how to use jq.

Learn a basic package manager such as apt or yum. You will create and delete many instances in this course, and how to “bootstrap” an instance automatically upon creation. This means loading software, and that means a package manager. One great way to practice this is with Docker (see below).

Be familiar with Python3. Know how to create virtual environments using pipenv, how to install dependencies, and manage local files, env variables, etc. within your code. Python2 will not work here. Bonus: Create an AWS account of your own and start working with the boto3 library.

Understand source control using Git and collaborative coding within GitHub.

Learn the basics of Docker. Know how to pull, create, and run containers.
